Команды администратора /broadcast — создать рассылку всем подписчикам. Мастер по шагам спросит: текст → фото/видео (можно «Пропустить») → кнопки (можно «Пропустить») → когда отправить (сейчас или по расписанию). /cancel — прервать создание рассылки. /stats — число подписчиков и их список (id, username). /scheduled — список запланированных рассылок. /cancel_scheduled <id> — отменить запланированную рассылку по её номеру. /help — это сообщение. Формат кнопок По одной в строке: Текст - https://ссылка Формат даты (мск) 24.06.2026 19:00 — полная дата 24.06 19:00 — текущий год 19:00 — сегодня
